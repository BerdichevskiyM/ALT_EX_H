# ALT_EX_H
Альтернативный экзамен ДМиТИ.

Тема: Тело кватернионов. Расширение системы компьютерной алгебры-7.

Выполнил студент гр. 4381, Бердичевский Максим.

Эта программа является продолжением коллоквиума ДМиТИ (https://github.com/Ozixnet/Colloquium-DMITI). Добавлен новый блок модулей H для работы с кватернионами.

Структура:
1) Папка modules содержит всю бизнес-логику, т.е. вычислительные модули.
2) Файл main.py - точка входа в приложение. Запустите этот файл, чтобы программа начала работу.
3) Файл tests.py содержит тесты, проверяющие все модули из папки modules. Запустите этот файл (через терминал: python tests.py), чтобы проверить корректность работы программы.
